# Number Guessing Game - Java
A fun  number guessing game built in Java.

## Features
- 3 difficulty levels: Easy (1-100, 10 attempts), Medium (1-500, 8 attempts), Hard (1-1000, 5 attempts)
- Hints: "Too high" or "Too low"
- Simple win/lose feedback

## How to Run
1. Compile  
   ```bash
   javac Main.java
